# re-dispatch

一个及其轻量的状态管理库，支持在`React`和`Angular`项目中使用，核心实现与框架无关。

`esbuild` + `terser`打包仅有 1017B（**不足 1KB**）

看看状态管理的实现代码，其实逻辑很简单……

只是原本非常简单的问题太容易被复杂化了。
